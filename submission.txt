Submission Dicoding Data Engineer - Studi Kasus Fashion Studio

Konteks:
Saya berperan sebagai data engineer pada perusahaan retail fashion dan design. Pipeline ini dibuat untuk mengambil data produk kompetitor dari Fashion Studio, membersihkan data agar siap dianalisis oleh tim data science, lalu menyimpan hasil akhirnya ke repositori data.

# Menjalankan skrip ETL pipeline
python3 main.py

# Menjalankan unit test pada folder tests
python3 -m pytest tests

# Menjalankan test coverage pada folder tests
coverage run -m pytest tests
coverage report -m

# Output utama
products.csv
